# 🎓 Sania AI Learning Buddy

Sania AI Learning Buddy is an AI-powered educational assistant built using **Python**, **Streamlit**, and **Google Gemini API**. It helps students understand concepts, learn through real-life examples, generate quizzes, and ask subject-related questions in a simple and interactive way.

## ✨ Features

- 📘 Explain concepts in simple language
- 🌍 Provide real-life examples
- ❓ Generate multiple-choice quizzes
- 💬 Ask questions on any topic
- ⚡ User-friendly Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- pyngrok
- Google Colab

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Sania-AI-Learning-Buddy.git
   cd Sania-AI-Learning-Buddy
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Add your Google Gemini API key.

4. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```

## 📸 Demo

Run the application using Streamlit and access it through the generated local or ngrok URL.
